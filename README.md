# Lin-Paper 📚
 
AI 智能数学教育系统 - 错题管理与变式题生成
 
## 功能特性 
- 🖼️ 图片识别：上传数学题目图片自动识别 
- 📝 错题本：保存错题到数据库 
- 📄 变式题试卷：生成 PDF 试卷 
- 📐 通用图形绘制：支持坐标系、椭圆、抛物线等 
 
## 快速开始 
1. 克隆仓库 
2. 安装依赖：`pip install -r requirements.txt` 
3. 配置 .env 文件 
4. 运行：`python main.py` 
 
## 技术栈 
- qwen-max / qwen-vl-plus / DeepSeek 
- Streamlit / ReportLab / Matplotlib
